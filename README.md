# 🚀 HabitFlow

Um aplicativo mobile minimalista para rastreamento de hábitos e construção de rotinas sólidas. Desenvolvido do zero para explorar conceitos de persistência de dados, UI/UX e gerenciamento de estado.

## 🎨 Identidade Visual (Neon Flow)
Escolhemos uma estética **Dark Mode** com contrastes vibrantes para foco e modernidade.

| Função | Elemento | Hexadecimal |
| :--- | :--- | :--- |
| **Fundo** | Background | `#121212` |
| **Cards** | Surface | `#1E1E1E` |
| **Destaque** | Primary (Roxo) | `#BB86FC` |
| **Check-in** | Accent (Ciano) | `#03DAC6` |
| **Texto 1** | High Emphasis | `#FFFFFF` |
| **Texto 2** | Medium Emphasis | `#A0A0A0` |

## 📂 Estrutura de Pastas
O projeto segue uma arquitetura modular e organizada:
* `assets`: Recursos visuais (ícones, imagens).
* `core`: Configurações globais, temas e estilos neon.
* `data`: Camada de persistência (Banco de dados local).
* `features`: Funcionalidades principais (Lista, Calendário).
* `models`: Estruturas de dados (O que é um Hábito).
* `services`: Lógica de notificações e armazenamento.
* `ui`: Telas (screens) e componentes reutilizáveis.
